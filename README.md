# Sourcey gap audit: Gotify REST API

This repository contains a reproducible audit of Sourcey 3.6.3 against the
public Gotify REST API Swagger snapshot pinned at commit
`8221c0b895fdd1b1b84c0d5784e9422eb314ecb5`.

- `report.md` — human-readable findings and recommendations
- `evidence.json` — captured commands, counts, examples, and classifications
- `receipt.json` — governed runx receipt
- `verification.json` — receipt verification result
- `X.yaml` / `run.mjs` — reproducible governed audit runner
